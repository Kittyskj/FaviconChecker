# FaviconChecker
Eeasy tool configured like telegram bot to check Favicon hash in md5

## Installation Linux
```bash
# After cloning change api and name to your
git clone https://github.com/Kittyskj/FaviconChecker \
 && cd FaviconChecker \
 && pip install -r requirements.txt \
 && python3 favcheck.py
```
